# MCP NestJS Samples

Sample projects demonstrating [@rekog/mcp-nest](https://github.com/rekog/mcp-nest) - a NestJS wrapper for building MCP (Model Context Protocol) servers.

## Available Samples

### [Pizzaz OpenAI Apps SDK](pizzaz-openai-apps-sdk/README.md)

Demonstrates building interactive ChatGPT widgets using `@rekog/mcp-nest`.

## About @rekog/mcp-nest

[@rekog/mcp-nest](https://github.com/rekog/mcp-nest) provides NestJS decorators for building MCP servers, making it natural for NestJS developers to:

- Define MCP tools with `@Tool()` decorator
- Expose MCP resources with `@Resource()` decorator
- Use Zod schemas for type-safe parameters
- Leverage standard NestJS dependency injection
